# Intelligent-cpu-scheduler
An interactive web-based simulator that demonstrates core CPU scheduling algorithms used in Operating Systems.

#Features
- Implemented Scheduling Algorithms:
  - First Come First Serve (FCFS)
  - Shortest Job First (SJF)
  - Priority Scheduling
  - Round Robin (RR)

- User-defined process inputs:
  - Arrival Time
  - Burst Time
  - Priority

- Dynamic Gantt Chart Visualization

- Performance Metrics:
  - Waiting Time (WT)
  - Turnaround Time (TAT)
  - CPU Utilization

#Technologies Used

- HTML (Structure)
- CSS (Styling & UI Design)
- JavaScript (Logic & Algorithms)

# How It Works

1. User inputs process details.
2. Selects scheduling algorithm.
3. Simulator computes execution order.
4. Displays:
   - Gantt Chart (visual timeline)
   - Average Waiting Time
   - Average Turnaround Time
   - CPU Utilization

# Objective

This project helps in understanding how CPU scheduling algorithms work internally and how they impact system performance.

# Note

This project is developed as part of Operating Systems coursework to demonstrate scheduling techniques and performance analysis.
